# Whereami

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/3zcurdia/whereami_rb/tree/master)

Ipinfo service, with basic memory cache

## Setup

Install dependencies

```
bundle install
```

Run service

```
rackup
```
